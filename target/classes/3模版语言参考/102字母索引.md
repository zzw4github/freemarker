# 字母索引

注意：

从FreeMarker 2.3.23开始，你可以使用camel case而不是all-lower-case作为指令名，`noParse` 而不是`noparse`。但是要知道，在同一模板中，FreeMarker将对所有作为模板语言一部分的标识符强制使用驼峰案例（用户定义的名称不受影响）。
