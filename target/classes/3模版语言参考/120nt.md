## 概要

```
<#nt>
```

## 描述

“没有修剪”。该指令禁用发生它的行中的[空白空间](https://freemarker.apache.org/docs/dgui_misc_whitespace.html#dgui_misc_whitespace_stripping)。它还禁止在同一行中存在的其他修剪指令的效果（效果`t`，`rt`， `lt`）。
